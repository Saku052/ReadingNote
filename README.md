## Reading Note

IOS application which tracks your reading habit and helps you visually understand your consistancy with charts.

### Key Features

- **reading time histogram**: check the histogram to see how many minutes you usually spend reading. 
- **implemented stop watch**: use the implemented stop watch to store data. 
- **Slider**: modern uder interface available to input minutes of reading manualy. 

### Technical Highlights

- **SwiftUI for UI Development**: Leveraging Swift UI framework to create modern interface.
- **Core Data**: Utilizing core data for saving customer data.
